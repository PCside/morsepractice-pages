モールス練習機 - Morse Code Trainer

CW（モールス通信）入門者のための練習アプリです。聞き取りはKoch法、打鍵は「画面全体が電鍵」+ 符号樹で練習できます。

よくある質問

  • 音が出ません — 本体側面のマナーモード（消音スイッチ）をご確認ください。本アプリの音量はマナーモードに従います。
  • 次の文字が解禁されません — 受信練習で1セッション（20問）を正答率90%以上で完了すると、次の文字が解禁されます。
  • 購入を復元したい — 設定タブ →「購入の復元」をお試しください。
